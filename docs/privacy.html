Yantrik Mind — Privacy Policy

Last updated: June 2026

Yantrik Mind is a personal, single-user AI companion operated privately by its owner for their own use. It is not a public service and does not offer accounts to other people. This policy explains what data the application accesses, how it is used, and how it can be deleted.

Who this covers

The only person whose data this application accesses is its owner and operator, who explicitly authorizes access using their own credentials. The application runs in development mode and does not collect, receive, or process data belonging to any other person.

What data we access

When the owner connects a Facebook account, the application may access — using an access token the owner generates and provides themselves — the owner's own:

  • Basic public profile information (name, profile ID) via the public_profile permission.
  • The owner's own posts and timeline activity via the user_posts permission.
  • The account's primary email address, if the email permission is granted.

How the data is used

Accessed data is used solely to provide personalized assistance to the owner — for example, to help the owner recall and reason over their own activity. The data is stored only in the owner's own private database on the owner's own server infrastructure.

What we do NOT do

  • We do not sell, rent, or share this data with any third party.
  • We do not use the data for advertising, profiling of other people, or any commercial purpose.
  • We do not transfer the data to any external service beyond what is required to display it to the owner.

Data retention

Data is retained only as long as it is useful to the owner and is stored on private, access-controlled infrastructure. The owner may remove any or all stored data at any time.

Data deletion

To delete data associated with this application:

  1. Revoke the application's access at any time from Facebook Settings → Business Integrations. This stops all further access immediately.
  2. To request deletion of data already stored, contact developer@pranab.co.in. Stored data will be permanently removed from the owner's private database.

Contact

Questions about this policy: developer@pranab.co.in.
